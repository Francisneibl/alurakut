# #ImersaoReact #Alura 

## Projeto Alurakut :octocat:
Iniciado da Imersão React da Alura, a ideia foi de recriar a interface do Orkut, e uitlizar dados da API do GitHub para obter dados de usuarios, e o DatoCMS para persister dados.
